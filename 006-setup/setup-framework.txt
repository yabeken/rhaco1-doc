* setup.php

rhaco はセットアップフレームワークとも呼ばれています．これは，rhaco に附属している setup.php から，モデルの生成，データベース操作，テスト等々開発に必要な作業を GUI で提供しているからです．

実際にどのような機能があるのか，見てみます．

** settings

アプリケーション設定を行います．次の設定が行えます．

:アプリケーションの初期処理:mod_rewrite の設定を記述した .htaccess を作成する．
:ログ設定:ログの出力や表示の設定
:キャッシュ:テンプレートやフィードのキャッシングの切り替え．有効期限の設定を行う．
:データベースの設定:project.xml で記述したデータベースの設定を行う．
:パス:アプリケーションの設置箇所や URL の設定を行う．
:全体:rhaco の設置パスやインストールサーバのパスを確認する．

** database

簡易データベース管理ツール

+ project.xml からテーブルの作成
+ 上記の SQL ダウンロード
+ project.xml に記述した初期データの作成
+ レコードの CRUD
+ query 発行
+ テーブル削除

** i18n

国際化ツール

+ pot の作成
+ pot から rhaco 用の PHP ファイル作成

あまりわかってない今日この頃．

** test

アプリケーションのテストを実行する．

** document

アプリケーションのクラスライブラリのドキュメントを表示．

** phpinfo

>|php|
phpinfo();
||<

の出力結果を見る．

** rhacodoc

rhaco 自身のドキュメントを表示

APIリファレンスについては， [[http://api.rhaco-users.jp]] もご覧ください．


** setup member

setup.php を実行するのに認証をかける．

** member

即席認証用のメンバー管理

** install

arbo，jaja をインストールする．
